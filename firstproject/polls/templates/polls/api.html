Здесь вы можете воспользоваться API
API
/api/v1/simbollist/ - получение списка всех монет
/api/v1/"simbol"/"interval"/ - получение объема монеты за выбранный интервал , обновление объемов монеты происходит в 02 секунду каждой минуты.
Типы параметра interval - "1_min", "5_min", "10_min", "30_min", "1_h", "3_h", "5_h"
Все возможные параметры simbol можно получить /api/v1/simbollist/
Пример рабочей ссылки - /api/v1/ethusdt/10_min/
/api/v1/"table"/ - Получение таблиц JSON
Типы параметра table :
table - данные для "обычной" таблицы
tableV2 - данные для "улучшенной" таблицы
Пример рабочей ссылки - /api/v1/table/
Параметры таблицы :
"asks" - Крупная лимитная заявка на покупку
"bids" - Крупная лимитная заявка на продажу
"name" - Название монеты
"timea" - Скорость в минутах за которую может быть разъедена заявка на покупку
"timea" - Скорость в минутах за которую может быть разъедена заявка на продажу
"rengea" - Расстояние до лимитной заявки в процентах на покупку
"rengeb" - Расстояние до лимитной заявки в процентах на продажу
"values" - Объем монеты за прошедшую минуту (в монетах)
"count_trades" - Колличество сделок по монете за минуту
